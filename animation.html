// window.innerHeight = 브라우저 안쪽 높이 // window.outerHeight = 브라우저 바깥쪽 높이 // window.innerWidth = 브라우저 안쪽 넓이 // window.outerWidth = 브라우저 바깥쪽 넓이 // window.screenTop / window.screenY = 모니터 상단에서부터 브라우저 상단의 위치 // window.screenLeft / window.screenX = 모니터 좌측에서부터 브라우저 좌측의 위치 // document.documentElement.offsetWidth = 브라우저에 보이는 넓이 // document.documentElement.offsetHeight = 브라우저에 보이는 높이(스크롤바 O) // document.documentElement.scrollHeight = 화면에 보이지 않는 부분까지 전체 높이 // document.documentElement.clientHeight = 화면에 보이는 높이(스크롤바 X) // document.documentElement.scrollTop = 전체 높이중 페이지가 스크롤되어 올라간 만큼의 높이
